# 🔥 **FORGE UI — FIGMA PROTOTYPE WIRING GUIDE**  
### *Scene‑by‑scene, interaction‑by‑interaction, variant‑by‑variant.*

---

# 🧱 1. **Page Setup (Figma Structure)**

Create a Figma page named:

```
Forge Prototype
```

Inside it, create frames:

```
00_Loading
01_ControlSurface
02_ConfigSurface
03_DiagnosticsOverlay
04_PresetOverlay
05_ErrorAlert
06_MonitorPreviewOverlay
```

Each frame uses **Auto Layout** and references only components from your Forge library.

---

# 🔥 2. **Loading → Control Surface Transition**

### Wiring
- Select the **Loading Screen frame**  
- Prototype tab → **After Delay: 2400ms**  
- Navigate to → **01_ControlSurface**  
- Animation: **Smart Animate**  
- Easing: `cubic-bezier(0.3, 0.8, 0.2, 1)`  
- Duration: **600ms**

### What this creates
- Ember ignition → coil charge → banner activation  
- UI panels slide in like steel plates  
- Scrubber drops with forge impact  

---

# 🟦 3. **Hover Interactions (Heat‑Blue Rim Sweep)**

For every button, monitor frame, terminal, slider:

### Wiring
- Component → Variant: `hover`  
- Trigger: **While Hovering**  
- Action: **Change To → Hover Variant**  
- Animation: **Smart Animate**  
- Duration: **180ms**  
- Easing: **Ease Out**

### What this creates
- Heat‑blue rim sweep  
- Steel bevel brightening  
- 1px lift  

---

# 🟧 4. **Press Interactions (Ember Compression)**

For all slab buttons and terminals:

### Wiring
- Trigger: **On Press**  
- Action: **Change To → Pressed Variant**  
- Animation: **Smart Animate**  
- Duration: **80ms**  
- Easing: **Ease In**

### Release
- Trigger: **On Release**  
- Action: **Change To → Active or Default**  
- Duration: **120ms**  
- Easing: **Ease Out**

---

# 🔥 5. **Active States (Ember Pulse Loop)**

For Play, Pause, Active Preset, Active Monitor:

### Wiring
- No prototype trigger needed  
- The variant itself contains a **looping animation** (Figma’s “After Delay → Change To Self”)  
- Delay: **0ms**  
- Duration: **1400ms**  
- Easing: **Ease In Out**

### What this creates
- Breathing ember glow  
- Heat‑treated steel pulse  

---

# 🟫 6. **Timeline Interactions**

### Scrubber Drag
- Trigger: **While Dragging**  
- Action: **Move**  
- Constraints: horizontal only  
- Add Smart Animate for tension effect

### Marker Add
- Trigger: **On Click**  
- Action: **Open Overlay → Marker Menu**  
- Animation: **Smart Animate**  
- Duration: **150ms**

### Marker Remove
- Trigger: **On Click**  
- Action: **Change To → Removed Variant**  
- Animation: **Smart Animate**  
- Duration: **200ms**

---

# 🟦 7. **Monitor Selection**

### Hover
- While Hovering → Hover Variant

### Select
- On Click → Change To Active Variant  
- Smart Animate (250ms, ease‑out)

### Error
- On Click (invalid state) → Change To Error Variant  
- Smart Animate (120ms, ease‑in)  
- Then → After Delay 200ms → Return to Default

---

# 🟧 8. **Audio Routing Terminals**

### Hover

### Press
- On Press → Pressed Variant

### Active
- On Click → Active Variant

### Warning / Error

- Use “Change To” with Smart Animate
